## 📌 الميزات (Features)

- ✅ البحث في **19+ منصة** (Instagram, GitHub, YouTube, Reddit, Twitter/X, LinkedIn, TikTok, وغيرها).  
  Search on **19+ platforms** (Instagram, GitHub, YouTube, Reddit, Twitter/X, LinkedIn, TikTok, and more).
- ✅ **دقة عالية** مع تحقق متعدد الطبقات.  
  **High accuracy** with multi-layer verification.
- ✅ **حفظ النتائج** بصيغة JSON.  
  **Save results** as JSON.
- ✅ **تأخير قابل للتعديل** (`--delay`) لتجنب الحظر.  
  Adjustable **delay** to avoid blocking.
- ✅ **مهلة قابلة للتعديل** (`--timeout`) لتجنب انتهاء الوقت.  
  Adjustable **timeout** to avoid timeouts.
- ✅ **دعم WHOIS** للنطاقات.  
  **WHOIS support** for domains.
- ✅ واجهة جميلة.  
  Nice interface.
